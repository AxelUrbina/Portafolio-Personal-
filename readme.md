
1. cómo correrlo (Live Server / python http.server)

Para correrlo en Live Server solo se da clic derecho al archivo index y se elige la primera opción - Para correr con python se usa el comando:  python -m http.server


2. qué hace el sitio y qué interacciones incluye

El sitio trae apartados de logros, proyectos y certificaciones. En proyectos se cargan datos y se muestran en tabla, también se puede buscar información en el campo de texto, se puede cambiar de sección y da mensajes como: Sin resultados o Error dependiendo el caso.
